# Phantom Engine  

A ligthweight modular game engine written in C for linux terminal games.  

## Overview  

Phantom Engine is designed to create terminal-based games using ANSI rendering  
The goal is to provide a simple but powerful archictecture that can be reused for:  

- Tower Defense games
- Roguelikes
- RPGs
- Simulation games
- Creepy experiences

## Features  

Current version : 0.0.1  

- ANSI terminal rendering
- Cursor control
- Terminal abstraction
- Basic color suport

### Planned  

- Frame Buffer
- Input System
- Entity system
- Scene management
- Physics
- Save system

## Philosophy

Phantom Engine focuses on:  

- Modularity
- Simplicity
- Learning
- Reusability
